# 📚 Livros Amazon

Projeto acadêmico desenvolvido em Java com foco em lógica de programação e Programação Orientada a Objetos (POO).

## 🚀 Funcionalidades
- Leitura de dados a partir de arquivo CSV
- Organização de informações de livros
- Estruturação de dados em classes
- Execução via classe principal

## 🛠️ Tecnologias
- Java
- Programação Orientada a Objetos (POO)

## 📂 Estrutura do Projeto
- `Book.java` → Representa os dados dos livros
- `DatasetReader.java` → Responsável pela leitura do arquivo CSV
- `Driver.java` → Classe principal que executa o sistema
- `Data.csv` → Base de dados dos livros

## ▶️ Como executar
1. Abra o projeto no IntelliJ
2. Execute a classe `Driver.java`

## 🎯 Objetivo
Praticar conceitos de lógica de programação, manipulação de dados e organização de código em Java.
